Visual Studio Code.

Visual Studio es un programa de desarrollo, tiene varios tipos de aplicaciones como Visual Studio que es utilizado mas para desarroyar programas, Visual Studio Code que es utilizado mas para programar codigo. En este caso usamos Visual Studio Code que es el que hemos utilizado para hacer el codigo de los websites.

Hay varios editores de codigo como Sublime entre otros, pero el mas practico es Visual Studio Code ya que tien plugins que nos haran mas practico el diseñar nuestro siteo web.

Codigo para hacer este sitio.

Para este sitio web se han usado los siguientes codigos; HTML y CSS. HTML se encarga de agregar las variables, los textos y mas, html es usado mas para agregar cosas al sitio web. CSS es usado para darle color, animacion, decorar y mas en el sitio web. En HTML se crean los tabs, los textos y mas, en CSS se les da propiedades como tipo de texto, tamaño, color, a las variables se les da tamaño para crear ventanas, color, opacidad entre otros.
